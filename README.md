Project Overview
 Modern dashboard application designed for students preparing for high school and university entrance exams. It integrates academic tracking, personal development, and social learning into one cohesive platform. Our goal is to enhance student focus, provide in-depth performance analysis, and foster a motivating study environment.

The project features a welcoming Landing Page that introduces the application's core benefits before directing users to the main Dashboard area for full access.

    Key Features
Landing Page & Secure Access: An aesthetic Landing Page to onboard users, followed by secure Login / Sign Up authentication systems.

Dark/Light Mode: Fully integrated theming optimized for comfortable studying both day and night.

    Advanced Focus Tools:

Promodoro Timer: Features different work presets and tracks consecutive successful sessions.

Study Vibe: Lofi Chill Spotify integration, plus an expanded ambient sounds library (e.g., Rain, Coffee Shop).

    In-Depth Academic Analytics:

Mock Exam Analysis: Net score trend charts, comparison with target scores, and critically, Topic-Based Error Tagging.

Subject Tracking (Spaced Repetition): Tracks topic completion and implements a system to alert the user when a subject is Due for Review (based on spaced repetition principles).

Reports: Detailed statistics showing time allocation by subject and identification of the most common error topics.

    Personalization & Social Network:

Profile Management: Ability to manage personal information, profile picture, and school details.

My Friends: Features for adding friends, managing requests, and comparing anonymous average net scores with the friend group.

Achievements (Gamification): Digital badges awarded upon reaching specific study milestones.
